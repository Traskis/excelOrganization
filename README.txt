Employee Data Analysis

Files:
- analyze_employees.py – Python script for data analysis
- Employees.xlsx – Input Excel file with employee data
- Analysis_Results.xlsx – Generated report with department counts, top earner, and high earners

How to use:
1. Make sure you have Python installed with the required packages:
   pip install pandas openpyxl

2. Run the script in your terminal:
   python main.py

3. Open 'Analysis_Results.xlsx' to view the results.
